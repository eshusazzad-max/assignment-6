
# DigiTools Platform

This is a simple website made with React and Tailwind CSS.

In this project, I created a digital tools platform where users can see products, add them to cart, and check pricing.

---

## Features

- Show products
- Add to cart
- Remove from cart
- Pricing section
- Steps section
- Footer

---

## Technologies

- React
- Tailwind CSS
- Vite

---

## How to run

1. Install dependencies:
npm install

2. Run project:
npm run dev
